# typora用strCodeテーマ

## Typora オフィシャルサイト

[https://typora.io/](https://typora.io/)

## コマンド

- `gulp`
  - SCSS ビルド
- `gulp convertFonts`
  - `ttf` 形式を `woff2` 形式にコンバート
- `gulp copyFonts`
  - `woff2` 形式フォントを `dist` フォルダにコピー
